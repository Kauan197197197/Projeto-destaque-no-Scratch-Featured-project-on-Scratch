
 Pixel Bound

Pixel Bound é um jogo de plataforma criado no Scratch que combina simplicidade visual, precisão e desafio.
Controle o personagem, um pequeno herói pixelado que precisa saltar com cuidado entre espinhos e armadilhas em um mundo minimalista.

Jogue agora: https://scratch.mit.edu/projects/1228971694/

Recursos Principais

Movimento fluido e física de salto ajustada

Obstáculos variados com padrões diferentes

Códigos organizados e modulares para fácil manutenção

Estrutura do Projeto

personagem: controle principal, movimento e colisão

espinhos: obstáculos que causam falha

nuvens e sol: elementos visuais dinâmicos que compõem o cenário

Objetivo

Criar um jogo de plataforma simples e envolvente, demonstrando conceitos de física, lógica de colisão e design minimalista no Scratch.
O projeto serve tanto como uma experiência divertida quanto como exemplo técnico de desenvolvimento visual.

Como Jogar

Use as setas do teclado para mover o personagem

Salte sobre os espinhos no momento certo

Alcance o final do percurso sem tocar em nenhum obstáculo
